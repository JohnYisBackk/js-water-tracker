# 💧 Water Tracker 

A modern hydration tracking application built with HTML, CSS, and JavaScript.

Track your daily water intake, set hydration goals, monitor progress in real time, and build healthier habits with an interactive water drop visualization.

---

## ✨ Features

- Add water intake with quick action buttons
- Add custom water amounts
- Daily hydration goal management
- Animated water drop progress visualization
- Hydration percentage tracking
- Water intake history log
- Delete individual entries
- Reset daily progress
- Local Storage persistence
- Mobile-friendly responsive design

## 📊 Functionality

### Water Tracking

Users can:

- Add 250ml, 500ml, or 750ml instantly
- Enter custom water amounts
- Track total daily intake
- Visualize hydration progress

### Daily Goals

Users can:

- Set a custom hydration goal
- Automatically update progress percentage
- Prevent progress from exceeding 100%

### History System

Every water intake creates a history entry:

- Amount consumed
- Time recorded
- Delete individual entries

### Data Persistence

All data is stored locally using Local Storage:

- Current water intake
- Daily goal
- Hydration history
